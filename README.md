# Дневник тренировки — Telegram Web App

Мини-приложение для Telegram: вставляешь план тренировки от тренера, удобно вбиваешь
подходы (вес × повторения), смотришь видео упражнений, работаешь с суперсетами — и
получаешь готовый текст в том же формате, чтобы отправить тренеру.

## Возможности

- Вставка сообщения тренера и автоматический разбор упражнений, подходов, рекордов и суперсетов
- Ввод подходов степперами `+/−` (быстро на телефоне), добавление/удаление подходов
- Переключение между упражнениями лентой-чипами; суперсеты подсвечены, заполненные — зелёные
- Кнопка «Видео» открывает ссылку упражнения прямо в Telegram
- Таймер отдыха (60/90/120 сек) с тактильным откликом
- Автосохранение прогресса (localStorage) — можно закрыть и вернуться
- Сборка итогового текста в исходном формате: копирование или отправка через Telegram
- Тема подстраивается под клиент Telegram (светлая/тёмная)

## Технологии

Чистый HTML + нативные ES-модули, без сборщика для продакшена — деплой статикой.
Внешние интеграции (Telegram WebApp SDK, localStorage) изолированы в `src/services/`.
Бизнес-логика (`src/core/`) не зависит от DOM и полностью покрыта тестами.

## Структура

```
src/
  core/        чистая логика (тестируется)
    parser.js      текст тренера -> модель
    serializer.js  модель -> текст тренеру
    sets.js        разбор/сборка подходов и заданий
    emoji.js       keycap-цифры (1️⃣ 2️⃣ …)
  services/    изоляция внешнего мира
    telegram.js    обёртка Telegram WebApp SDK
    storage.js     localStorage
  app/state.js  состояние + автосохранение
  ui/           экраны и компоненты
index.html
```

## Разработка

```bash
npm install
npm run dev        # локальный сервер Vite
npm test           # прогон тестов
npm run coverage   # тесты + покрытие (порог 70%)
```

## Деплой на GitHub Pages

1. Залить репозиторий на GitHub.
2. Собрать статику: `BASE_PATH=/имя-репозитория/ npm run build` — результат в `dist/`.
   (Либо выложить корень репозитория как есть — приложение работает без сборки.)
3. В настройках репозитория включить GitHub Pages на ветку/папку с файлами.
4. Полученный `https://<user>.github.io/<repo>/` указать в BotFather:
   `/newapp` → выбрать бота → задать URL мини-приложения.

## Как пользоваться

1. Открыть мини-приложение в Telegram.
2. Скопировать сообщение с тренировкой от тренера и вставить на первом экране.
3. Заполнить подходы по каждому упражнению.
4. На последнем шаге нажать «Скопировать текст» и отправить тренеру.
